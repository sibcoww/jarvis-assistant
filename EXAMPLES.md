# Примеры голосовых команд Jarvis Assistant

## 🗣️ Браузер и поиск

```
"открой браузер"          → Запустить Chrome
"перейди на google.com"   → Открыть Google
"открой сайт github.com"  → Открыть GitHub
"гугл python"             → Поиск "python" в Google
"поиск как готовить торт" → Поиск рецептов
```

## 🎵 Медиа-контроль

```
"включи музыку"    → Play/Resume
"пауза"            → Pause
"стоп"             → Stop
"далее"            → Next track
"следующая"        → Next track
"назад"            → Previous track
"предыдущая"       → Previous track
```

## 🔊 Громкость

```
"громкость 50"         → Установить громкость на 50%
"сделай громче"        → +10%
"сделай громче на 20"  → +20%
"сделай тише"          → -10%
"убавь громкость на 5" → -5%
```

## 📅 Календарь и время

```
"какая дата"       → "Сегодня 1 марта 2026 года"
"сегодня дата"     → Показать текущую дату
"какое время"      → "Текущее время: 23 часа 34 минуты"
"который час"      → Показать время
```

## 📝 Заметки и напоминания

```
"запомни купить молоко"           → Добавить заметку
"запомни позвонить маме завтра"   → Сохранить заметку
"напоминание встреча в 15:00"     → Создать напоминание
"вспомни"                         → Прочитать последние 5 заметок
"прочитай заметки"                → Показать заметки
```

Файлы сохраняются в:
- Заметки: `~/.jarvis/notes.json`
- Напоминания: `~/.jarvis/reminders.json`

## 📂 Приложения

```
"открой браузер"     → Chrome
"запусти vscode"     → Visual Studio Code
"открой telegram"    → Telegram Desktop
"открой блокнот"     → Notepad
```

## 📁 Файлы и папки

```
"создай папку Проект"      → Создать папку
"создай папку Documents"   → Создать папку
```

## 🎯 Сценарии

```
"рабочий режим"  → Запустить сценарий (открыть VS Code + браузер)
```

Настройка сценариев в `src/jarvis/config.json`:
```json
{
  "scenarios": {
    "рабочий режим": ["open:vscode", "open:browser"]
  }
}
```

## 💡 Советы

1. **Активация:** Скажите "Джарвис" для активации ассистента
2. **Команда:** После активации произнесите команду
3. **TTS:** Включите `enable_tts=True` для озвучивания ответов
4. **Синонимы:** Можно использовать различные варианты команд
5. **Логи:** Все действия записываются в консоль

## 🔧 Настройка

### Включить TTS
```python
from src.jarvis.executor import Executor
ex = Executor(enable_tts=True)
```

### Добавить приложение
В `src/jarvis/config.json`:
```json
{
  "apps": {
    "spotify": "C:/Users/User/AppData/Roaming/Spotify/Spotify.exe"
  },
  "synonyms": {
    "музыка": "spotify",
    "спотифай": "spotify"
  }
}
```
