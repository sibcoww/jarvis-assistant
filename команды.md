# Команды голосового ассистента Jarvis

Этот файл содержит полный список команд, которые поддерживает проект на текущий момент.

## 1) Браузер и веб

- `перейди на <сайт>`
- `открой сайт <сайт>`
- `гугл <запрос>`
- `поиск <запрос>`

## 2) Приложения

### Открыть приложение

- `открой браузер` / `запусти хром`
- `открой телеграм`
- `открой ватсап`
- `открой vscode`
- `открой блокнот`
- `открой <что-то>` (общий паттерн)

### Закрыть приложение

- `закрой браузер`
- `закрой телеграм`
- `закрой ватсап`
- `закрой vscode`
- `закрой блокнот`

## 3) Громкость

- `громкость 50`
- `звук 70`
- `сделай тише`
- `убавь громкость на 20`
- `сделай громче`
- `добавь громкость на 15`

## 4) Медиа

- `включи музыку`
- `пауза`
- `далее`
- `назад`

## 5) Презентация

- `следующий слайд`
- `слайд вперед`
- `предыдущий слайд`
- `слайд назад`
- `запусти презентацию`
- `останови презентацию`

## 6) Окна (Windows)

- `сверни окно`
- `разверни окно`
- `закрой текущее окно`
- `переключи окно`
- `окно влево`
- `окно вправо`
- `окно вверх`
- `окно вниз`
- `раздели экран пополам`
- `два окна рядом`

Примечание: для команды `раздели экран пополам` текущий активный экран ставится влево, после чего пользователь выбирает второе окно в Snap Assist справа.

- `создай папку <имя>`
- `сделай папку <имя>`

- `напоминание <текст>`
- `напомни через 10 минут <текст>`
- `напомни в 19:30 <текст>`

## 10) Таймер

- `таймер на 10 секунд`
- `таймер на 5 минут`
- `таймер на 1 час`
- `сколько осталось`
- `отмени таймер`

## 11) Погода

- `погода`
- `какая погода`
- `какая погода в астане`
- `что с погодой в алматы`

## 12) To-Do (задачи)

- `добавь задачу <текст>`
- `запиши мне в дела <текст>`
- `покажи задачи`
- `список задач`
- `выполнил задачу 1`
- `отметьте как первая задача`
- `удали задачу 1`
- `удали задачу <фрагмент>`

## 13) Быстрые системные действия

- `выключи компьютер`
- `выключи пк`
- `выключи нахуй комп` (поддерживается)
- `перезагрузи компьютер`
- `перезапусти комп`
- `режим сна`
- `заблокируй экран`

Для рискованных действий (`shutdown/restart/sleep`) обязательно голосовое подтверждение:

- `подтверждаю` — выполнить
- `отмена` — отменить

### Сохранение фактов

- `запомни ...`
- `запомни что ...`

- `сохрани ...`
- `учти что ...`
- `имей в виду ...`
- `запиши себе что ...`

### Просмотр фактов

- `что ты помнишь обо мне`
- `что ты знаешь обо мне`

### Удаление фактов

- `забудь всё обо мне`
- `забудь последнее`
- `забудь <фрагмент>`
- `забудь эту информацию обо мне`

- `покажи историю`
- `покажи контекст`

## 17) Уточняющие шаги (когда команда неполная)

Если команда слишком общая, ассистент задает уточняющий вопрос:

- `открой сайт` -> спросит какой сайт
- `открой приложение` -> спросит какое приложение
- `громкость` -> спросит конкретный уровень
- `найди` -> спросит что искать
