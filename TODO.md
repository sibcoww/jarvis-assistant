## 9. Интеграция ChatGPT + озвучка eSpeak NG (Опциональная функция)

### Архитектура
**Цель:** Добавить умные ответы через ChatGPT API с озвучкой через eSpeak NG, сохраняя полный оффлайн-режим по умолчанию.

### План реализации

#### 9.1. Конфигурация
- [ ] Добавить секцию `chatgpt` в `config.json`:
  - `enabled: bool` — включить/выключить ChatGPT (по умолчанию `false`)
  - `api_key: str` — OpenAI API ключ (хранить зашифрованно или в переменной окружения)
  - `model: str` — модель GPT (например, `gpt-4o-mini`, `gpt-3.5-turbo`)
  - `max_tokens: int` — лимит токенов ответа (по умолчанию 150-200)
  - `temperature: float` — креативность ответа (0.7-1.0)

### Примеры работы системы

#### Быстрые примеры ответов (готовые оффлайн-фразы)

| Действие | Ответ |
|---|---|
| Открытие приложения | "Открываю браузер" |
| Установка громкости | "Громкость установлена на 50 процентов" |
| Неизвестная команда | "Я не понял команду" |
| Активация | "Слушаю" |

#### Пример 1: Оффлайн-режим (по умолчанию)
**Ситуация:** ChatGPT выключен в настройках (`config.json`: `"chatgpt": {"enabled": false}`)

**Сценарий:**
```
Пользователь: "Джарвис, открой браузер"
→ NLU распознаёт intent: open_app (приложение: браузер)
→ Executor запускает браузер
→ Лог: "✅ Открываю браузер"
→ Никакие сетевые запросы не отправляются
```

**Сценарий с неизвестной командой:**
```
Пользователь: "Джарвис, какая погода в Москве?"
→ NLU не распознаёт intent (confidence < 0.65)
→ Executor запускает оффлайн fallback-озвучку
→ eSpeak говорит: "Я не понял команду. Повторите вопрос"
→ Лог: "🗣 Оффлайн fallback: не понял команду"
```

#### Пример 2: ChatGPT включен, обработка вопроса
**Ситуация:** ChatGPT включён (`"chatgpt": {"enabled": true}`), eSpeak включён

**Сценарий:**
```
Пользователь: "Джарвис, какая погода в Москве?"
→ NLU не распознаёт intent (confidence < 0.65)
→ Executor передаёт запрос в ChatGPT fallback
→ ChatGPTClient отправляет запрос в OpenAI API:
   {
     "model": "gpt-4o-mini",
     "messages": [
       {"role": "system", "content": "Ты голосовой ассистент Джарвис. Отвечай кратко и по делу."},
       {"role": "user", "content": "какая погода в Москве?"}
     ],
     "max_tokens": 150
   }
→ OpenAI возвращает: "Извините, у меня нет доступа к актуальной информации о погоде. Рекомендую проверить погодный сайт или приложение."
→ EspeakTTS озвучивает ответ: espeak-ng -v ru -s 150 "Извините, у меня нет доступа..."
→ Лог: "🤖 ChatGPT: Извините, у меня нет доступа к актуальной информации..."
→ GUI показывает ответ на главной вкладке
```

```

```

    intent, confidence = self.nlu.parse(text)
    
    if confidence >= 0.65:
    # Стандартная команда + готовая оффлайн-озвучка
        self._handle_standard_intent(intent)
    self._speak_offline_template(intent)
    elif self.config["chatgpt"]["enabled"]:

            self.espeak_tts.speak(response)
    else:
    # Оффлайн fallback
    self.espeak_tts.speak("Я не понял команду. Повторите вопрос")
