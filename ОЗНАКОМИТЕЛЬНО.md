# Jarvis Assistant — быстрый старт

## Что это за программа

`Jarvis Assistant` — голосовой помощник для Windows.  
Он слушает команду, распознает речь, выполняет действие на компьютере и озвучивает ответ.

Проект сделан для дипломной работы: просто, понятно и с упором на реальные сценарии.

## Что умеет

- открывать программы и сайты;
- запускать пользовательские сценарии (цепочки действий);
- вести список задач, таймеры и напоминания;
- управлять окнами, громкостью, медиа и презентацией;
- отвечать на вопросы через AI fallback, если команда неизвестна;
- хранить базовую память о пользователе.

## Что нужно, чтобы работало

Минимум:

1. **Windows 10/11**
2. **Python 3.11+**
3. Установленные зависимости из проекта

Для полного режима:

- **OpenAI API key** — для AI fallback;
- **Picovoice key** — если хочешь wake-word через Porcupine;
- микрофон по умолчанию в системе.

## Первый запуск

1. Запусти приложение (`python -m gui.app`).
2. Открой `Настройки`.
3. Выбери микрофон.
4. При необходимости включи AI и сохрани ключ OpenAI.
5. Нажми `Старт`.

После запуска скажи wake-word и команду.

## Примеры команд для старта

- `открой телеграм`
- `запусти сценарий стрим`
- `добавь задачу доделать диплом`
- `таймер на 10 минут`
- `покажи задачи`
- `окно влево`

## Как устроен fallback

- Сначала Jarvis пытается понять команду локально (быстро и стабильно).
- Если команда нестандартная, подключается AI fallback.
- Для системных опасных действий есть подтверждение (`подтверждаю` / `отмена`).

## Настройки, которые стоит проверить

- `Микрофон` — устройство и чувствительность.
- `Распознавание и активация` — движок wake-word, таймауты.
- `Озвучка (TTS)` — скорость и громкость.
- `AI (искусственный интеллект)` — включение и модель.
- `Программы / Сайты / Сценарии` — управление пользовательскими списками.

## Если что-то не работает

- Перепроверь выбранный микрофон и ключи API.
- Нажми `Перезагрузить конфиг`.
- Для добавленных программ проверь путь к `.exe`.
- Если ответ странный — `Очистить контекст`.
