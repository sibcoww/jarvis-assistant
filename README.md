# Jarvis Assistant

Голосовой ассистент для Windows: офлайн-распознавание речи, локальное выполнение команд, AI fallback для сложных запросов, память о пользователе и GUI на PySide6.

## Что умеет

- Офлайн ASR на Vosk (русская модель).
- Wake-word: `Джарвис` (Porcupine или текстовый режим).
- Continuous режим: после команды слушает следующие команды без повторного wake-word.
- Команды:
  - громкость (`сделай тише`, `громкость 35`, уточнения при неполной фразе),
  - браузер и поиск,
  - запуск приложений,
  - базовые заметки и напоминания.
- AI unified pipeline:
  - `reply` (ответ текстом),
  - `action` (структурированная команда для программы).
- Память о пользователе:
  - чистка разговорного шума,
  - дедупликация близких фактов,
  - команды управления памятью.
- Безопасность:
  - подтверждение рискованных файловых действий (`delete_file`, `move_file`).

## Быстрый старт

### 1) Установка

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

### 2) Модель Vosk

Скачайте русскую модель Vosk и распакуйте в:

`models/vosk-model-ru-0.42`

Источник: [https://alphacephei.com/vosk/models](https://alphacephei.com/vosk/models)

### 3) Запуск GUI

```bash
python -m gui.app
```

## Настройки

Основной конфиг: `src/jarvis/config.json`

Ключевые секции:

- `audio`:
  - `phrase_timeout` — максимум длины фразы,
  - `silence_timeout` — пауза, после которой фраза считается завершенной.
- `ai`:
  - `enabled`,
  - `model`,
  - контекстные лимиты.
- `apps`, `synonyms`, `sites` — маршрутизация локальных команд.

## Примеры команд

- Браузер:
  - `открой сайт` -> ассистент уточнит, какой сайт.
  - `найди в браузере` -> ассистент уточнит, что искать.
- Громкость:
  - `поставь громкость` -> уточнение: на какую громкость.
  - `сделай тише` / `сделай громче` -> уточнение: на сколько.
- Память:
  - `покажи факты обо мне`,
  - `удали факт <фраза>`,
  - `очисти временную память`,
  - `что ты знаешь обо мне`.

## Архитектура (кратко)

- `src/jarvis/engine.py` — основной цикл, ASR/NLU/executor orchestration.
- `src/jarvis/executor.py` — выполнение команд, AI fallback, pending-уточнения, безопасность.
- `src/jarvis/unified_ai_turn.py` — единый формат ответа AI (`reply`/`action`).
- `src/jarvis/memory_store.py` — профиль и факты пользователя.
- `gui/app.py` — десктоп GUI.

## Для дипломной демонстрации

Рекомендуется показать сценарии:

1. Обычный вопрос (`reply`) vs действие (`action`).
2. Pending-уточнение на неполной команде (`открой сайт`, `поставь громкость`).
3. Память без дублей + управление памятью.
4. Подтверждение рискованного действия.
5. PIPE-логи: `text -> intent -> result`.
